# ⚽ World Cup: Statistical Match Modeling

### Project Overview
Analysis of historical FIFA World Cup data to identify statistical patterns in match outcomes and team performance. This project applies scientific methods to sports analytics.

### Tech Stack
* **Core Science:** Advanced Statistics — *Curriculum by Stanford University*.
* **Analysis:** Exploratory Data Analysis (EDA) in Python & SQL.
* **Visualization:** Matplotlib & Seaborn for impact-driven data storytelling.

### Key Findings (Systems Thinking)
* **Goal Distribution:** Modeled the probability of scoring across different tournament stages.
* **Performance Trends:** Quantified regional dominance and home-field advantage factors.

### Project Structure
* `data/`: Historical World Cup match and player datasets.
* `images/`: Statistical charts and performance visualizations.
* `notebooks/`: `world_cup_statistical_modeling.ipynb`.
* `reports/`: [World_Cup_Statistical_Insights.pdf](./World_Cup_Statistical_Insights.pdf).
